# QUANTUM STACK（仮）

複数の盤面候補が同時に分岐していく、ブラウザー向けの重ね合わせ落ち物パズルです。

2種類のミノをひとつの重ね合わせ状態として操作し、着地した瞬間にそれぞれの形を置いた世界へ分岐します。どこかの世界でラインが完成すると、その条件を満たした世界だけが残り、候補が収束します。

## 遊び方

- `←` / `→`：左右へ移動
- `↓`：ソフトドロップ
- `Z`：左回転
- `X` または `↑`：右回転
- `Space`：着地させて世界を分岐
- `Esc`：一時停止

左側には全候補を重ねた未確定盤面、中央には個別の候補盤面が表示されます。中央の候補を選ぶと、左側でその盤面を拡大確認できます。確認しても世界は確定しません。

## ゲームのルール

1. 色の異なる2種類のミノを同時に操作します。
2. 着地すると、それぞれのミノを置いた盤面候補へ分岐します。
3. ラインが完成した候補がある場合、その候補だけが残ります。
4. 置ける候補がなくなるとゲームオーバーです。
5. 状態数が50万を超えた場合は、ブラウザー保護のため強制的にゲームオーバーになります。

## ローカルでの起動

Node.js 22.13.0以降を使用してください。

```bash
npm install
npm run dev
```

本番用ビルドの確認は次のコマンドで行えます。

```bash
npm run build
```

## 技術構成

- TypeScript
- React
- Canvas 2D
- Web Worker
- vinext / Vite

多数の候補盤面を扱えるよう、盤面データには型付き配列を使用し、分岐とライン消去の計算をWeb Workerで実行しています。

## プライバシー

このゲームは、プレイヤーの氏名、メールアドレス、位置情報などの個人情報を収集・保存しません。
